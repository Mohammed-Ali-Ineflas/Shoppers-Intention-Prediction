# Online Shoppers Purchasing Intention - Machine Learning Project

## 📌 Description
Ce projet a pour but de prédire l'intention d'achat des visiteurs d'un site e-commerce en se basant sur leurs comportements de navigation. 

## 📊 Dataset
Les données utilisées proviennent du fichier `online_shoppers_intention.csv`. Elles incluent des métriques telles que le temps passé sur les pages (Administratives, Informationnelles, Liées au produit), le Bounce Rate, l'Exit Rate, etc.

## 🛠️ Outils & Modèles
- **Langage :** Python
- **Librairies :** Pandas, Scikit-Learn, Seaborn, Matplotlib
- **Modèles de Machine Learning :**
  - Régression Logistique (avec données normalisées - MinMaxScaler)
  - Random Forest Classifier (avec données originales)

## 🚀 Comment exécuter le projet
1. Cloner ce repository :
   `git clone https://github.com/Mohammed-Ali-Ineflas/Shoppers-Intention-Prediction.git`
2. Installer les dépendances :
   `pip install -r requirements.txt`
3. Lancer le notebook Jupyter `online_shoppers_model.ipynb`.
