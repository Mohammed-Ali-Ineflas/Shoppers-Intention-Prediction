# Online Shoppers Purchasing Intention - Machine Learning Project

## 📌 Description
Ce projet a pour but de prédire l'intention d'achat des visiteurs d'un site e-commerce en se basant sur leurs comportements de navigation. Il permet de comprendre les facteurs clés qui poussent un visiteur à finaliser une transaction.

## 📊 Dataset
Les données utilisées proviennent du fichier `online_shoppers_intention.csv`. Elles incluent des métriques telles que :
- Le temps passé sur les pages (Administratives, Informationnelles, Liées au produit)
- Le Bounce Rate (Taux de rebond)
- L'Exit Rate (Taux de sortie), etc.

## 🛠️ Outils & Modèles
- **Langage :** Python
- **Librairies :** Pandas, Scikit-Learn, Seaborn, Matplotlib
- **Modèles de Machine Learning implémentés :**
  - Régression Logistique (avec données normalisées - `MinMaxScaler`)
  - Random Forest Classifier (avec données originales)

## 📈 Résultats
 "Le modèle Random Forest a surpassé la régression logistique avec une meilleure précision pour identifier les acheteurs potentiels."
